Norsk legemiddelhåndbok search system.

This is a search system that uses patient notes as search query for therapy chapters of the Norwegian handbook for pharmaceutical interventions (Legemiddelhåndboka).

Requirements

It require Python 3.2 and Whoosh 2.3.2 (or later versions).

Background

This project is our solution to the mandatory assignment in the course TDT4215 Web-intelligence at Norwegian University of Science and Technology (NTNU) spring 2012. Assignment text and our deliverables can be found on the projects download page.

Creators

We are @eventh, @snarby and @weilin (group 1).
